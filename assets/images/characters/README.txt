Pour ajouter des images de personnages aux boutons numériques :

1. Placez vos images dans ce dossier
2. Nommez les fichiers comme suit :
   - 0.png pour le chiffre 0
   - 1.png pour le chiffre 1
   - 2.png pour le chiffre 2
   - etc.
   - point.png pour le point décimal

Les images doivent être au format PNG avec un fond transparent pour un meilleur rendu.

Conseils :
- Taille recommandée : 200x200 pixels
- Format : PNG avec transparence
- Noms des fichiers : 0.png, 1.png, 2.png, etc.
